
# NOUB Sports Platform - V2.0

"Community sports identity platform for amateur players" - "منصة الهوية الرياضية المجتمعية للاعبين الهواة "
## Architecture
- **Frontend:** HTML5, CSS3 (Modular), Vanilla JS (ES6 Modules).
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions).
- **Environment:** Telegram Mini App (TMA).

## Directory Structure
- `assets/`: Images, Fonts, Sounds.
- `css/`: Modular Styles (Base, Components, Views).
- `js/`: Application Logic (MVC Pattern).
